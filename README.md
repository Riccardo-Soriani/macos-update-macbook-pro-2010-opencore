## What this guide covers

This guide focuses on the **initial steps** required before OpenCore can even be installed — the part that's rarely documented online. Specifically, how to go from a completely isolated macOS 10.8.5 system to a state where OpenCore Legacy Patcher can run and create a bootable USB installer.

The full upgrade path covered:

```
macOS 10.8.5
    → Firefox 48 (to regain internet access)
    → macOS El Capitan 10.11 (to unlock the system)
    → macOS High Sierra 10.13 (OpenCore minimum requirement)
    → OpenCore Legacy Patcher → macOS Ventura
```
